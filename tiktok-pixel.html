🔧 TikTok Pixel ID Gir

📋 Pixel ID'yi TikTok Ads Manager'dan alın

ℹ️ Nasıl Kullanılır?

1. TikTok Ads Manager'a git
2. Assets → Events → Web Events
3. Pixel oluştur veya mevcut olanı seç
4. Pixel ID'yi kopyala ve yukarıya yapıştır
5. Kaydet butonuna bas

✅ PageView - Sayfa görüntüleme

✅ SubmitForm - Form gönderme
